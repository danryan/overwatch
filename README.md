* How does a data point enter the system?




A datapoint comes in from the client through the API, and looks something like this:


{
  :data_point => {
    :key => "key",
    :value => "value",
    :metric => "key",
    :node => "uuid"
  }
}


or (load average)

{
  :data_point => {
    :key => "One Minute",
    :value => "1.01",
    :metric => "load-average",
    :node => "71140ea02837012e15ee0025bcdd0280"
  }
}
